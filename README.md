# NextStep English  

Este projeto propõe o desenvolvimento de um site para apoiar o ensino de inglês em uma escola de pequeno porte. A plataforma busca melhorar a comunicação entre professores e alunos e incentivar o aprendizado contínuo fora da sala de aula por meio de recursos interativos e educativos.

Entre as funcionalidades do site estão tarefas diárias de prática, ranqueamento de alunos, quizzes interativos, chatbot educacional, fórum de conteúdos e uma aba de notícias em inglês, permitindo que os alunos pratiquem o idioma de forma mais dinâmica e conectada com temas do mundo real.

## Funcionalidades

### 👨‍🎓 Para alunos
- Realizar quizzes interativos
- Acompanhar ranking de desempenho
- Receber tarefas diárias (Daily Tasks)
- Acessar notícias em inglês
- Praticar com chatbot educacional

### 👨‍🏫 Para professores
- Criar quizzes personalizados (Quiz Maker)
- Acompanhar desempenho dos alunos
- Gerenciar conteúdos e atividades

### 🌐 Sistema geral
- Interface interativa e responsiva
- Sistema de ranking (gamificação)
- Integração com notícias em inglês
- Jogos educativos (ex: Agro’s Friends)

## Arquitetura e Tecnologias

O sistema segue o padrão de arquitetura em camadas:
- Controller → recebe requisições
- Service → regras de negócio
- Repository → acesso ao banco
- Model → entidades

Frontend e backend são separados.

### Backend (Java + Spring Boot)
Desenvolvimento da API REST 
Gerenciamento de regras de negócio 
Integração com banco de dados

### Frontend (React)
Interface interativa e responsiva 
Comunicação com backend via API (fetch/axios) 
Atualização dinâmica dos dados 

### Banco de Dados (PostgreSQL)
Armazenamento persistente dos dados 
Estrutura relacional (clientes, carros, etc.) 
Garantia de integridade e segurança
